This is a GitHub repository of data testing QED-TDDFT implementations in Python on two dye molecules, Coumarin 343 and NBD-PC. 
For more information on the project, please see the paper and presentation uploaded.  
Within each folder, a README.md file can be found, detailing the different jobs submitted.
